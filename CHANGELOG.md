# Changelog

Todos los cambios notables del proyecto se documentan en este archivo.

## [1.0.0] - 2026-02-05

### Añadido

- Tarjeta de tarea (Task Card) con cabecera: nombre del proyecto, última actualización, pill "Link a la tarjeta".
- Bloque **Descripción** con texto editable.
- Bloque **Fechas**: Inicio, Para validar, Aprobado, Handoff (todos editables).
- Bloque **Contactos** con "Añadir contacto"; cada contacto: nombre, rol, email (editables) y opción de eliminar.
- Bloque **Recursos** debajo de contactos con "Añadir recurso"; lista editable con placeholder `[recurso]`; primera fila solo eliminable si tiene texto o hay más de una fila.
- Bloque **Checklist** (lista de tareas) con "Añadir tarea"; cada ítem con checkbox, label y texto secundario editables.
- Footer con avatar del usuario y badge de estado (WIP, REVIEW, APPROVED, BLOCKED, ARCHIVED, HANDOFF) seleccionable desde el menú del widget.
- Frames de texto editable nombrados `editable-text` para consistencia en Figma.
